    Grid Layout

    4KB

    Grid layout system created for building modern web layouts easily.

    • General usage
    • Container
    • Responsive columns
    • Fractions
    • Nesting grids
    • Rows
    • Gap
    • Reverse
    • Ordering
    • Alignment
    • Common patterns

    Pros

    • Lightweight. 4KB vs 15KB (Bootstrap grid system)
    • Simple to use
    • Custom system to our needs. Flexible
    • Easy reverse and ordering
    • Less code. Simple DOM.
    • Use of standard and modern code
    • RTL direction compatible
    • Can be used in combination with other layout systems (Flex, cols system). It's not exclusive.

    Cons

    • Not supported in IE11

    Want to know more about grid layout? Here's a complete guide

    General usage

    • The columns are defined in the parent component.
    • The grid classes can be applied to the .container element.
    • Firefox dev tools work amazingly for grid creation.
    <div class="container grid-2">
  <div>...</div>
  <div>...</div>
</div>

    Ordering is applied to the child item.
