Several utility classes are available for spacing, using <code>.{property}-{sides}-{size}</code> naming.

## Property options
<ul>
  <li><code>m</code> – for classes that set <code>margin</code></li>
  <li><code>p</code> – for classes that set <code>padding</code></li>
</ul>

## Sides
<ul>
  <li><code>a</code> – for classes that set the <code>margin</code> or <code>padding</code> on all sides of the element</li>
  <li><code>t</code> – for classes that set the <code>top</code> side of the property</li>
  <li><code>r</code> – for classes that set the <code>right</code> side of the property</li>
  <li><code>b</code> – for classes that set the <code>bottom</code> side of the property</li>
  <li><code>l</code> – for classes that set the <code>left</code> side of the property</li>
  <li><code>x</code> – for classes that set both <code>left</code> and <code>right</code></li>
  <li><code>y</code> – for classes that set both <code>top</code> and <code>bottom</code></li>
</ul>

## Sizes
<ul>
  <li><code>0</code> – for classes that eliminate the <code>margin</code> or <code>padding</code> by setting it to <code>0</code></li>
  <li><code>1</code> – for classes that set the <code>margin</code> or <code>padding</code> to <code>space-8</code> (8px)</li>
  <li><code>2</code> – for classes that set the <code>margin</code> or <code>padding</code> to <code>space-16</code> (16px)</li>
  <li><code>3</code> – for classes that set the <code>margin</code> or <code>padding</code> to <code>space-24</code> (24px)</li>
  <li><code>auto</code> – for classes that set the <code>margin</code> to auto</li>
</ul>
<p>And only for vertical cases:</p>
<ul>
  <li><code>4</code> – for classes that set the <code>margin-top, margin-bottom</code> or <code>padding-top, padding-bottom</code> to <code>space-32</code> (32px)</li>
  <li><code>5</code> – for classes that set the <code>margin-top, margin-bottom</code> or <code>padding-top, padding-bottom</code> to <code>space-40</code> (40px)</li>
</ul>

## Responsive sections

These are the sizes of larger, responsive spacing used for separating sections of a page:

<ul>
  <li><code>section-1</code> classes provide <strong>16px, 24px or 32px </strong> of spacing depending on screen width</li>
  <li><code>section-2</code> classes provide <strong>32px, 48px or 64px </strong> of spacing depending on screen width</li>
  <li><code>section-3</code> classes provide <strong>48px, 72px or 96px </strong> of spacing depending on screen width</li>
</ul>

Spacing classes are available for each of the <strong>vertical</strong> utility classes at each section size e.g.
